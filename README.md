# EEG_Biometrics
Library for EEG Biometrics project for HERO Lab - UC Irvine

# Requirements

Python == 3.7 or 3.8
tensorflow == 2.X (verified working with 2.0 - 2.3, both for CPU and GPU)
keras >= 2.8.0

sklearn >= 1.0.2
scikit-learn >= 0.20.1
matplotlib >= 2.2.3
pandas >=  1.4.1
numpy >= 1.22.0
